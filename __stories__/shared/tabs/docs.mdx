# Tabs

<br />

## Overview

`Tabs` 컴포넌트를 사용하면 탭 형태의 네비게이션을 구현할 수 있습니다.

이 컴포넌트는 사용자가 탭을 클릭할 때마다 해당 탭과 연관된 콘텐츠 패널을 표시합니다.

각 탭 버튼은 `Tabs.TabList` 내에 위치하고, 각 콘텐츠 패널은 `Tabs.TabPanels` 내에 위치합니다.

이 두 요소는 순서에 따라 서로 매칭됩니다. (ex. 첫 번째 탭 버튼은 첫 번째 콘텐츠 패널과 연결)

<br />

## Usage

```tsx
<Tabs>
  <Tabs.TabList>
    <Tabs.Tab>Tab 1</Tabs.Tab>
    <Tabs.Tab>Tab 2</Tabs.Tab>
    <Tabs.Tab>Tab 3</Tabs.Tab>
  </Tabs.TabList>
  <Tabs.TabPanels>
    <Tabs.TabPanel>탭 내용 1</Tabs.TabPanel>
    <Tabs.TabPanel>탭 내용 2</Tabs.TabPanel>
    <Tabs.TabPanel>탭 내용 3</Tabs.TabPanel>
  </Tabs.TabPanels>
</Tabs>
```

활용 예시는 Example을 통해 확인 가능합니다.

<br />

## Tabs

설명: Tab 기능을 사용하기 위해서는 최상단에는 Tabs 컴포넌트로 감싸야 합니다. 이를 통해 내부의 Tabs 아이템들이 서로 연동됩니다.

<br />

## Tabs.TabList

설명: Tabs.TabList는 탭 버튼 목록을 정의합니다. 각 Tabs.Tab 컴포넌트가 이 목록 안에 위치하며, 사용자가 클릭할 수 있는 탭을 제공합니다. 탭 버튼과 Tabs.TabPanels 내의 콘텐츠 패널은 순서에 따라 매칭됩니다.

<br />

## Tabs.Tab

설명: Tabs.Tab은 실제 탭 버튼을 정의합니다. 이 버튼을 클릭하면 해당 탭과 관련된 콘텐츠 패널이 표시됩니다.

### Props

```tsx
export interface ITabProps extends ComponentPropsWithoutRef<'li'> {
  /** Tab 안의 내용 */
  children: ReactNode;
  /** 그 외 커스텀 스타일 */
  etcStyles?: CSSObject;
}
```

<br />

## Tabs.TabPanels

설명: Tabs.TabPanels는 탭 콘텐츠 패널을 포함하는 컨테이너입니다. 각 Tabs.TabPanel 컴포넌트가 이 컨테이너 안에 위치하여 탭 선택에 따라 표시됩니다. Tabs.TabList의 탭 버튼과 순서가 매칭됩니다.

<br />

## Tabs.TabPanel

설명: Tabs.TabPanel은 각각의 탭과 관련된 콘텐츠 패널을 정의합니다. Tabs.Tab과 연동되어 선택된 탭의 콘텐츠를 표시합니다.

### Props

```tsx
export interface ITabPanelProps extends ComponentPropsWithoutRef<'div'> {
  /** TabPanel 안의 내용 */
  children: ReactNode;
  /** 그 외 커스텀 스타일 */
  etcStyles?: CSSObject;
}
```
