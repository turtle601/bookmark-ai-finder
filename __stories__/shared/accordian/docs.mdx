# Accordian

<br />

## Overview

Accordion 컴포넌트를 사용하면 컨텐츠를 숨기거나 표시할 수 있는 아코디언 스타일의 UI를 구현할 수 있습니다.
동일한 ID를 가진 버튼과 패널이 서로 연동되어, 버튼을 클릭할 때 해당 패널의 내용을 토글(열기/닫기)할 수 있는 아코디언 UI 컴포넌트를 구현합니다.

<br />

## Usage

```tsx
<Accordion.Provider>
  <Accordion.Button id={'1'}>
    <Accordion.Icon id={'1'} size={12} strokeWidth="12" />
    <Text label={'아코디언 스타일'} />
  </Accordion.Button>
  <Accordion.Panel id={'1'}>이거는 아코디언 패널 내용</Accordion.Panel>
</Accordion.Provider>
```

활용 예시는 **Example**을 통해 확인 가능합니다.

<br />

## Accordian.Provider

설명: 아코디언 컴포넌트는 반드시 Accordion.Provider로 감싸야 합니다. 이를 통해 내부의 아코디언 아이템들이 서로 연동됩니다.

<br />

## Accordian.Button

설명: Accordian.Panel을 열고 닫는 버튼을 정의합니다. id 속성으로 패널과 연결할 고유한 ID를 지정해야 합니다.

<br />

### Props

```tsx
export interface IAccordianButtonProps
  extends ComponentPropsWithoutRef<'button'> {
  /** Accordian.Button의 id, mapping된 Accordian.Panel을 열고 닫을 수 있다 */
  id: string;
  /** Accordian.Button 안의 내용 */
  children: React.ReactNode;
  /** 그 외 커스텀 스타일 */
  etcStyles?: CSSObject;
}
```

## Accordian.Panel

설명: Accordian.Button과 연동되는 컴포넌트로, 해당 ID를 가진 버튼을 클릭하면 열리거나 닫히는 영역입니다. id 속성으로 버튼과 연결할 고유한 ID를 지정해야 합니다.

### Props

```tsx
export interface IPanelProps extends ComponentPropsWithoutRef<'div'> {
  /** Accordian.Panel의 id, mapping 된 Accordian.Button 클릭 시 해당 Panel이 열린다 */
  id: string;
  /** Accordian.Panel 안의 내용 */
  children: React.ReactNode;
}
```

<br />

## Accordian.Icon

설명: Accordian.Button 내에 아이콘을 삽입할 수 있습니다. id는 버튼과 동일하게 설정해야 하며, size와 strokeWidth 등의 속성으로 아이콘의 크기와 스타일을 조정할 수 있습니다.

### Props

```tsx
export interface IIconProps {
  /** Accordian.Button의 id와 같은 id 값을 할당해야 합니다 */
  id: string;
  /** Accordian.Icon ">"의 크기를 나타냅니다 */
  size: number;
  /** Accordian.Icon ">"의 색상을 나타냅니다 */
  color?: string;
  /** Accordian.Icon ">"의 선 굵기를 나타냅니다 */
  strokeWidth?: string;
}
```
